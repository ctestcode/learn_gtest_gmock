ResultOf 函数结果 匹配，使用场景：对复杂的struct结构体，制定匹配函数

ResultOf作用是对一个函数的结果进行匹配检查。可以使用函数逻辑，对更复杂的struct结构进行匹配。

Matcher Description
ResultOf(f, m) f(argument) matches matcher m, where f is a function or functor.
