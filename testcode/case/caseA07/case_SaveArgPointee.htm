SaveArgPointee 动作，使用场景: 保存指针参数所指的值或结构数据
SaveArgPointee<N>(pointer) Save the value pointed to by the N-th (0-based) argument to *pointer.

SaveArgPointee关键字可以在MOCK结束后，保存指针参数的值。

下面的例子的作用是，期待CheckABC被调用后，把第一个指针参数的值保存在变量a里。
int a;
EXPECT_CALL(_mock,CheckABC)
.WillOnce(SaveArgPointee<1>(&a));
