Truly, 使用函数作为MATCHER

Matcher Description
Truly(predicate) predicate(argument) 返回值可以通过if 判定为 true,  predicate 可以是函数或伪函数 .

gMock 提供了一系列内建的 匹配器. 当你有自己的算法函数时，只要函数参数类型符合，可就以使用自己的函数作为算法匹配器. 举例：

int IsEven(int n) 
{ 
    return (n % 2) == 0 ? 1 : 0;
}

EXPECT_THAT(ret, Truly(IsEven))

提示：自定义的函数不必返回bool类型，只要可以使用这样判定就可以被使用 if (condition) ....
