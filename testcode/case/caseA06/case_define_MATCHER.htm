如何写一个匹配函数

提醒：请确保Matcher的功能是单一的，没有任何副作用，比如，不要去改变指针参数的值。

 定义MATCHER* 的语法如下 

 MATCHER(name, description_string_expression) { statements; }

定义了一个名字为name的匹配函数，会执行statement逻辑。匹配函数必须返回bool来表示匹配是否成功。statement范围里可以使用arg(参数)和 arg_type(参数类型)。

 description string 是注释，用来描述这个匹配的作用, 匹配失败的时候也输出这个信息，描述信息可以是空("")

For example:

 MATCHER(IsDivisibleBy7, "") { return (arg % 7) == 0; }
